# Fraud Guard Bot

* `Config.json` dosyasındaki bilgileri doldur.
* Consolu açıp ```npm install``` yazarak tüm modülleri kur.
* Kurulum bittikten sonra ```node fraud.js``` yaz ve botu başlat.


## Botun İntentlerini Açmayı Unutma!
